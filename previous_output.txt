 1) New Turkish game releases and developer announcements

   - A new game called "Kefi" has been announced by Teknogames, an independent Turkish game studio. Kefi is a rhythm-based action game set in a mystical world. (Source: GamesIndustry.biz)
   
   - "The Wild Eight", a survival action game developed by HypeTrain Digital and Eight Points, will have a new expansion called "Frostlanders". The expansion will be available on Steam on March 10th. (Source: SteamDB)
   
   - "Paleozoica", a survival management game developed by Pixel Federation, has released its latest update "The Great Migration". This update includes new creatures and challenges. (Source: IGN)
   
   - "Art of War: Red Tides", a real-time strategy game by Game Science, has been released globally. The game features Turkish language support. (Source: Steam Store)
